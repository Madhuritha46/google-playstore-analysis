## 📌 Overview

An advanced data analytics and machine learning dashboard built using Python and Streamlit to analyze Google Play Store applications, ratings, installs, pricing trends, and user sentiments.

---

## ✨ Features

- 📊 Data Cleaning & EDA
- 📈 Interactive Visualizations
- 🔍 Search & Filters
- 💬 Sentiment Analysis
- 🤖 ML Rating Prediction
- 📥 Download Filtered Data
- 🌙 Dark Theme Dashboard

---

## 🛠 Tech Stack

- Python
- Pandas
- NumPy
- Plotly
- Streamlit
- Scikit-learn
- TextBlob
